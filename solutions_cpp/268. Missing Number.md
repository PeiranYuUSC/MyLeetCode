## Gauss's Formula
## O(n), O(1)
```c++
class Solution {
public:
    int missingNumber(vector<int>& nums) {
        int sumNoMiss = (nums.size()+1) * nums.size() / 2;
        int sumNums = 0;
        for (int i = 0; i < nums.size(); i++) {
            sumNums = sumNums + nums[i];
        }
        
        return sumNoMiss - sumNums;
    }
};
```
