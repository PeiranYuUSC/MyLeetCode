## Use string as a stack
## O(n), O(n)
```c++
class Solution {
public:
    string removeDuplicates(string s) {
        string res = "";
        for (char letter : s) {
            if (letter == res.back()) {
                res.pop_back();
            } else {
                res.push_back(letter);
            }
        }

        return res;
    }
};
```
